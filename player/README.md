
# Environment variables

Build command:
```bash
npm run deploy
```

Build environment variables
```bash
WORKER_NAME=
D1_DB_NAME=
D1_DB_ID=
```
This environment variables are using to create `wrangler.toml` file and deploy the worker.

Runtime settings
```bash
site_name=
site_slogan=
id_offset=37951
footer_code=
fetch_api_url=
fetch_api_key=
```
